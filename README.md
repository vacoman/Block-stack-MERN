## Para intalar 

Abrir una terminal ubicado en: 

### `/backend/`

Ingresar los siguientes codigos:

### `npm install`

### `npm rund dev`

En otra terminal ubicado en:

### `/frontend/`

Ingresar los siguientes codigos:

### `npm install`

### `npm start`
